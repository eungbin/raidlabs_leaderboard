# RaidLabs leader_boad

### Stack
- html, css, typscript, react
- this project don't use css framework/library or other table frameworks.
- fetch api (call dummy data)

### Clone Project
``` bash
git clone https://github.com/eungbin/raidlabs_leaderboard
```

### install packages
``` bash
npm i
```

### Run Project
``` bash
npm run dev
```

### port
``` bash
http://localhost:5500
```
